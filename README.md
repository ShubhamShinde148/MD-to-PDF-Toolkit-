# MD to PDF Toolkit (MD Catalyst Pro)

A world-class, premium Markdown Studio designed for writing, live previewing, and exporting Markdown documents into beautifully formatted PDFs and web-ready HTML pages.

### 🚀 Key Features
* **Interactive Monaco Editor**: Integrated bracket matching, word wraps, syntax coloring, and quick formatting toolbar.
* **A4 Print View Simulation**: Live rendering simulating realistic printed margins, shadows, and fonts (WYSIWYG).
* **Syntax Highlighting**: Real-time coloring of code blocks matching VS Code and GitHub styles (powered by Highlight.js).
* **Flexible PDF Styling**: Choice between simple grayscale PDF layouts and colorful premium sheets with visual alerts.
